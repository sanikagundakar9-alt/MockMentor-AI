# MockMentor AI 🤖

MockMentor AI is an advanced AI-powered Interview Preparation Web Application designed to help students and job seekers improve their interview skills through mock interviews, AI chatbot interaction, voice input, resume upload, and performance analytics.

---

# Features ✨

- AI Mock Interview System
- HR & Technical Questions
- AI Chatbot
- Voice Recognition
- Resume Upload
- Performance Analytics
- Timer System
- Dark / Light Mode
- Typing Animation
- Responsive Dashboard UI

---

# Technologies Used 🚀

- HTML5
- CSS3
- JavaScript
- Chart.js
- Web Speech API
- Local Storage

# Functionalities 🧠

## Mock Interview
Generate interview questions and practice answers with a timer system.

## Voice Input
Answer interview questions using microphone voice recognition.

## AI Chatbot
Ask HR or technical interview questions and get instant responses.

## Resume Upload
Upload resumes for interview preparation.

## Analytics Dashboard
Visualize interview performance using charts and score analysis.

## Dark Mode
Switch between dark and light themes.
